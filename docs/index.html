Getting Started: Development

  1. Install dependencies at project root

    npm install

  2. Set up husky hooks

    npm run prepare

  3. Initialize MSW

    npm run msw:build

  4. Start Storybook server

    npm run storybook:run

  5. Run tests

    npm run test
